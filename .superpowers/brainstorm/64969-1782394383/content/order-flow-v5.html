确认图 → 合同 → 工艺单，严格递进

核心流程

📋 录入订单
填信息+传图片
→
🖼️ 生成确认图
花型+关键信息
→ 手工发给客户
→
✅ 客户确认
系统中标记已确认
→
📄 生成销售合同
自动生成
→
📝 生成工艺单
必须有合同才能创建

下推 → 从合同直接生成工艺单
📝 新建工艺单（两种方式）
方式一：从合同下推
在合同详情页点击「下推」→ 自动创建工艺单
→ 合同标记为「已下推」（不可重复下推）
方式二：新建时选择合同
在工艺单新建页面 → 下拉选择「未下推」的合同
→ 自动带入合同信息 → 补充工艺参数
→ 保存后合同自动标记「已下推」

数据库设计要点

-- 合同表需有下推状态标记
sales_contract {
id, order_id, contract_no, status, ...
is_pushed_down boolean DEFAULT FALSE,
process_sheet_id (nullable FK)
}

-- 工艺单表关联合同
process_sheet {
id, contract_id (FK NOT NULL), ...
}

-- 约束：工艺单创建时，contract_id 对应的合同
-- is_pushed_down = FALSE，创建后立即更新为 TRUE

📄 合同状态规则
📝 草稿
• 合同信息可任意修改
• 可反复生成确认图（V1, V2, V3...）
• 可软删除
• 不可下推工艺单
✅ 客户已确认
• 当前版本锁定（不可修改）
• 可下推生成工艺单
• 可软删除（需确认无已下发工艺单）
🔒 已下推
• 完全锁定
• 不可删除
• 关联工艺单已下发至车间
📝 工艺单状态规则
📄 保存
• 可编辑
• 可软删除
• 不下发
✅ 确认
• 不可编辑
• 可撤回至保存
• 可下发
🚀 下发
• 只读
• 合同变为「已下推」
• 合同不可删

完整状态流转图

// === 合同侧 ===
📝 合同草稿 ──→ (生成确认图V1) ──→ 📝 草稿 ──→ (修改) ──→ 📝 草稿 ──→ (生成确认图V2)
──→ (客户确认V2) ──→ ✅ 客户已确认
✅ 客户已确认 ──→ (下推工艺单) ──→ 🔒 已下推

// === 工艺单侧 ===
📄 保存 ──→ ✅ 确认 ──→ 🚀 下发
 │      ↑ ↕ 可撤回
 │      └ 校验合同版本号是否为最新
 └ 可软删除
